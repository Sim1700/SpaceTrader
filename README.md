# 🚀 Space Trader (Kosmiczny Kupiec) 🌌

A dynamic, single-screen space trading strategy game built with a neon cyberpunk aesthetic. Step into the shoes of a spaceship captain, navigate between planets, trade resources, manage your fuel, and survive unpredictable pirate attacks!

Developed as a modern web-app using **PHP Sessions**, **Tailwind CSS**, and **Vanilla JavaScript** for cinematic transitions.

---

## 🎮 Game Features

* **Ship Selection:** Choose between the nimble **Sokół** (high fuel, small cargo) or the heavy freighter **Gargantua** (low fuel, massive cargo).
* **Dynamic Economy:** Prices for *Water* and *Time Crystals* change dynamically depending on which planet you are currently visiting (Earth, Mars, or Cybertron).
* **Cinematic Travel Animations:** Immersive warp-drive transitions and screen-shake effects when moving between celestial bodies.
* **Random Space Events:** Experience unexpected anomalies, find lost cargo, or trigger a **RED ALERT** during sudden pirate ambushes.
* **Zero Database Architecture:** The entire game state, inventory, and player statistics are securely managed using native PHP Sessions (`$_SESSION`).

---

## 🛠️ Tech Stack

* **Backend:** PHP 8.x (State management & game logic)
* **Frontend:** Tailwind CSS (Cyberpunk neon UI & layout)
* **Interactions:** Vanilla JavaScript (Warp-drive animations, UI slide-ins, and audio/visual event handling)
* **Version Control:** Git & GitHub
